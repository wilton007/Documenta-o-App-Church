## Autenticação

* RF01 - O sistema deve permitir que o pastor (sub-admin) realize o cadastro de usuários.
* RF02 - O sistema deve permitir que o membro realize login utilizando seu nickname e sua data de nascimento.
* RF03 - O sistema deve manter a sessão do usuário até que ele faça logout.
* RF04 - O sistema deve permitir que o usuário edite seus dados cadastrais.


## Agenda

* RF05 - O sistema deve permitir visualizar a agenda anual da igreja.
* RF06 - O sistema deve exibir os detalhes de um evento.
* RF07 - O administrador deve poder cadastrar eventos.
* RF08 - O administrador deve poder editar eventos.
* RF09 - O administrador deve poder excluir eventos.


## Avisos

* RF10 - O sistema deve exibir os avisos cadastrados.
* RF11 - O administrador deve poder cadastrar avisos.
* RF12 - O administrador deve poder editar avisos.
* RF13 - O administrador deve poder excluir avisos.


## Pedidos de oração

* RF14 - O sistema deve permitir registrar pedidos de oração.
* RF15 - O sistema deve permitir registrar pedidos de oração de forma anônima.
* RF16 - O sistema deve permitir visualizar os pedidos de oração.
* RF17 - O administrador deve poder remover pedidos inadequados.


## Palavra do dia

* RF18 - O sistema deve permitir que membros publiquem uma reflexão.
* RF19 - O sistema deve permitir visualizar as reflexões publicadas.
* RF20 - O administrador deve poder remover reflexões inadequadas.


## Bíblia

* RF21 - O sistema deve permitir consultar a Bíblia.
* RF22 - O sistema deve permitir pesquisar livros, capítulos e versículos.
* RF23 - O sistema deve permitir pesquisar palavras na Bíblia.
* RF24 - O sistema deve permitir salvar versículos favoritos.


## Harpa Cristã

* RF25 - O sistema deve permitir consultar os hinos da Harpa Cristã.
* RF26 - O sistema deve permitir pesquisar hinos por número.
* RF27 - O sistema deve permitir pesquisar hinos por título.
* RF28 - O sistema deve permitir favoritar hinos.


## Cronograma

* RF29 - O sistema deve exibir o cronograma semanal da igreja.
* RF30 - O administrador deve poder alterar o cronograma.


## Perfil

* RF31 - O sistema deve exibir as informações do perfil do usuário.
* RF32 - O usuário deve poder alterar sua foto de perfil.
* RF33 - O usuário deve poder alterar seu telefone.


## Administração

* RF34 - O administrador deve gerenciar os usuários cadastrados.
* RF35 - O administrador deve alterar o cargo de um usuário.
* RF36 - O administrador deve ativar ou desativar usuários.


## Notificações

* RF37 - O sistema deve enviar notificações quando um novo aviso for publicado.
* RF38 - O sistema deve enviar notificações quando um novo evento for cadastrado.


## Favoritos

* RF39 - O sistema deve permitir favoritar versículos.
* RF40 - O sistema deve permitir favoritar hinos.
