## Tecnologia


* RNF01 - O backend será desenvolvido em Python utilizando FastAPI.
* RNF02 - O aplicativo será desenvolvido em React Native.
* RNF03 - O banco de dados utilizado será MariaDB.
* RNF04 - O ambiente de desenvolvimento utilizará Docker.


## Segurança

* RNF05 - O sistema deverá utilizar autenticação JWT.
* RNF06 - As credenciais dos usuários deverão ser armazenadas de forma segura utilizando algoritmos de hash.


## Usabilidade

* RNF07 - O aplicativo deverá possuir interface responsiva para diferentes tamanhos de tela.
* RNF08 - O aplicativo deverá funcionar em modo retrato.
